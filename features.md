# 🚀 Features

## Core Features
- Medicine scheduling (name, dosage, time)
- Smart alerts/notifications
- Action confirmation (Taken / Missed)
- Tracking & history logs

## Family Features (USP)
- Add family members
- Caregiver monitoring
- Missed dose alerts to family
- Family dashboard

## Accessibility Features
- Regional language support (Hindi, Marathi)
- Elderly-friendly UI (big buttons, simple layout)

## Awareness Module
- Educates users about importance of medicines

## Future Scope
- OCR prescription scanning
- Voice alerts
- IoT smart pill box
