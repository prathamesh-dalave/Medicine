# Product Requirements Document (PRD)

## Product Name
Smart Medicine Reminder System

## Objective
To improve medication adherence using reminders, tracking, and caregiver monitoring.

## Target Users
- Elderly patients
- Chronic patients
- Families

## Core Features
- Medicine scheduling
- Smart reminders
- Action confirmation (Taken/Missed)
- Tracking logs
- Family monitoring
- Missed dose alerts

## System Flow
Reminder → User Action → Log → If Missed → Notify Caregiver

## Tech Stack
- Frontend: HTML, CSS, JS
- Backend: Python (Flask)
- Database: PostgreSQL

## Constraints
- Simple UI (elderly-friendly)
- No complex frameworks
- Backend-ready architecture

## Future Scope
- OCR scanning
- Voice alerts
- IoT integration
