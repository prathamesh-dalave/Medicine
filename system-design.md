# 🧠 System Design

## Architecture

Frontend:
- HTML, CSS, JavaScript

Backend:
- Python (Flask)

Database:
- PostgreSQL

## Workflow

1. User adds medicine schedule
2. System triggers alert at time
3. User clicks "Taken" or "Missed"
4. If missed → notify caregiver
5. Data stored in database
6. Dashboard updates

## Key Logic

Reminder → Action → Tracking → Alert → Dashboard
